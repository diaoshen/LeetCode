## Account Related

#### Show git config 
``` git config --list ```
#### Show git username 
``` git config user.name ```

## Common 

#### Create local Repo 
``` git init ```
#### Create git ignore 
``` touch .gitignore ```

#### Add File(s) to stage 
``` git add <file1> <file2> <file3>  ```   or ``` git add . ``` to add all files.
#### Commit 
``` git commit -m "commit message" ```
#### Status of working tree 
``` git status ```

## Remote Repo 

#### Pull 
``` git pull ```
#### Pull origin master 
``` git pull origin master ```
#### Clone Repo 
``` git clone <link> ```
